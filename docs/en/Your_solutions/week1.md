# Physics Solutions

## Week 1

### Problem 1

Given:

a = [2, 1, -3]  
b = [4, -2, 1]

Find:

1) Magnitude of vectors  
2) Dot product  
3) Cross product  
4) Angle between vectors

Solution:

|a| = √(2² + 1² + (-3)²) = √14  
|b| = √(4² + (-2)² + 1²) = √21  

Dot product:  
a · b = (2)(4) + (1)(-2) + (-3)(1) = 3  

Cross product:  
a × b = [-5, -14, -8]

Angle between vectors:  
θ = arccos((a · b) / (|a||b|))  
θ = arccos(3 / √294) ≈ 79.9°
