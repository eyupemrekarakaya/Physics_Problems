# Solution 09 — Optimization Problem

## Problem

y = 3 − x²

Find the dimensions of the rectangle under the curve in the first quadrant with maximum area.

---

## Solution

Area:

A = x · y

Substitute y:

A = x(3 − x²)

A = 3x − x³

Derivative:

A' = 3 − 3x²

Set derivative to zero:

3 − 3x² = 0

x² = 1

x = 1

Height:

y = 3 − 1

y = 2

---

## Answer

Maximum rectangle:

width = 1  
height = 2  

Area = 2
