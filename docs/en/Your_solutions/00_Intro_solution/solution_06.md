# Solution 06 — Function Analysis

## Problem

f(x) = 3x² − 12x + 7

Identify any local maxima or minima.

---

## Solution

First derivative:

f'(x) = 6x − 12

Set derivative to zero:

6x − 12 = 0

x = 2

Second derivative:

f''(x) = 6

Since f''(x) > 0, the point is a minimum.

Calculate function value:

f(2) = 3(2²) − 12(2) + 7

f(2) = 12 − 24 + 7

f(2) = −5

---

## Answer

Local minimum at:

(2, −5)
