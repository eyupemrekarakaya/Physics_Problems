# Solution 05 — Trigonometry

## Problem

A vector A has magnitude:

|A| = 15

Angle with horizontal axis:

θ = 60°

Find the horizontal and vertical components.

---

## Solution

Horizontal component:

Ax = A cosθ

Ax = 15 cos(60°)

Ax = 15 × 0.5

Ax = 7.5

Vertical component:

Ay = A sinθ

Ay = 15 sin(60°)

Ay = 15(√3/2)

Ay ≈ 12.99

---

## Answer

Ax = 7.5
